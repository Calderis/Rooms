Pour lancer le projet en local : 
- le serveur php est classique, aucun besoin de quoi que ce soit (n'oubliez pas d'importer le fichier sql dans votre BDD)
- le serveur node : installer les dépendances avec "npm install" une fois que vous vous serez placé dans le dossier à travers votre console de commande , installez grunt en suivant la procédure normale ( "npm install -g grunt-cli" puis "npm install grunt-cli") et installez les dépendances qui posent problème si la console vous embète. Je les ai effacés pour que le projet pèse moins lourd.

Note : si "npm install" ne fonctionne pas, vérifiez que vous avez bien node d'installé sur votre pc
